# Homepage Components for KRYROS

All components are built using Next.js 14+ with TypeScript and Tailwind CSS.

## Components Created:

1. **PromoBanner.tsx** - Big full-width promo banner
2. **ProductPromoList.tsx** - 5 vertical product promo items
3. **ProductReviews.tsx** - Product reviews slider with customer images and purchased products
4. **DiscountBanner.tsx** - Discount banner with background image and overlay
5. **DualBannerSection.tsx** - Two side-by-side promo banners
6. **TrendProductsBanner.tsx** - Trend/promo banner with badges, bullet points, and image

## Usage:

Import the components into your homepage and pass the data from your CMS API!

```tsx
import PromoBanner from './HomepageComponents/PromoBanner';
import ProductPromoList from './HomepageComponents/ProductPromoList';
import ProductReviews from './HomepageComponents/ProductReviews';
import DiscountBanner from './HomepageComponents/DiscountBanner';
import DualBannerSection from './HomepageComponents/DualBannerSection';
import TrendProductsBanner from './HomepageComponents/TrendProductsBanner';

// Then use them in your homepage with data from CMS
```
